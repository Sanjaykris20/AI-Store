# 🛍️ AI E-Commerce Store - AI Recommendation Engine

AI-powered e-commerce platform with collaborative filtering, content-based filtering, and LLM-enhanced recommendations.

## 🚀 Quick Start

```bash
cd ai_ecommerce
pip install -r requirements.txt
python app.py
```

## Features
- Product recommendations (collaborative, content-based, hybrid)
- User cart, wishlist, checkout
- Chatbot integration
- Reflex/Streamlit UI

## Project Structure
```
ai_ecommerce/
├── app.py               # Main app
├── backend/             # Recommendation engines
├── pages/               # UI pages
├── state/               # App state management
├── ai_recommendation/   # LLM recommendations
└── requirements.txt
```

Visit: http://localhost:8501
